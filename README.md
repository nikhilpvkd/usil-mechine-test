## Project stucture

```src/
├── components/
│ └── Row.tsx
├── utils/
│ └── generateData.ts
├── App.tsx
└── main.tsx
```

### Installation

```
npm install
```

### Run Project

```
npm run dev
```

The app will be available at:http://localhost:5173

### Run server

```
node server.js
```

The server will run on http://localhost:3001
